# CityOps

CityOps is a desktop-only, browser-based transit network simulation game with a bus-first MVP scope.

## Quick start

### Prerequisites

- Node.js 22+
- pnpm 10+

### Install dependencies

```bash
pnpm install
```

### Run the web app

```bash
pnpm --filter @city-ops/web dev
```
